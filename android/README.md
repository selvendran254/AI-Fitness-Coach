# Android TWA / Health Connect (Feature #8)

This folder is a stub for a future **Trusted Web Activity** or **Capacitor** app to access:

- Health Connect (steps, heart rate, sleep)
- Background sync while the web app is closed

## Recommended next step

1. `npx @capacitor/cli init` in project root
2. Add `@capacitor/android` + Health Connect plugin
3. Point WebView to production URL (HTTPS required for Web Bluetooth + PWA)

Until then, use **Devices → Phone** tab (Health Connect / Google Fit) in the browser.
